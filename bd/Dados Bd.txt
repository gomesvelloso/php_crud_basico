Olá!

Para conseguir usar crud, você vai precisar o mysql instalado na sua máquina local
ou no seu servidor.

Este arquivo apagará qualquer banco de dados criado com o nome empresa e criará 
um novo banco de dados novo chamado empresa. Após o banco ser criado, ele criará
3 registros de usuário.

Como as senhas são criptografadas, você pode não saber qual é a senha, então
a senha do Usuário 01 é 01234567890.

Para logar, insira user01@teste.com e a senha 01234567890.

Caso precise alterar os dados de conexão com o banco de dados, por favor, abre a classe
Conexao.php (ela está dentro da pasta class). 
Lá você pode modificar o usuário, senha e nome da base de dados.
